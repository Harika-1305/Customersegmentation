# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project performs customer segmentation using the K-Means clustering algorithm. It groups customers based on their age, gender, and spending behavior to identify distinct customer segments.

## 📊 Dataset
- Dataset: Mall Customers Dataset  
- Features:
  - CustomerID  
  - Genre (Gender)  
  - Age  
  - Annual Income (k$)  
  - Spending Score (1–100)  

## ⚙️ Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  

## 🚀 Steps Performed
1. Imported required libraries  
2. Loaded dataset using Pandas  
3. Encoded categorical feature (Gender) using LabelEncoder  
4. Selected relevant features (Age, Gender, Spending Score)  
5. Applied K-Means clustering algorithm  
6. Assigned cluster labels to customers  
7. Visualized clusters using scatter plot  

## 🤖 Model Details
- Algorithm: K-Means Clustering  
- Number of Clusters: 5  
- Random State: 42  

## 📊 Visualization
- Scatter plot of Age vs Spending Score  
- Different colors represent different customer clusters
